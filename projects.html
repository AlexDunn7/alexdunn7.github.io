Tech Dream Website

Overview

Tech Dream is a fully functioning ecommerce business. It is developed through WordPress, with custom scripts to make the website function better using PHP. I created the social media accounts and I manage the content that is posted on there. Upon placing an order, there is an automated response that sends the information to the supplier and a notification to the email marketing system, that is set up via MailChimp. To increase the SEO score, there are also blog posts that are available on the website that were written by me. I am constantly updating the website and adding new features to help improve the business.

Skills Required

    • Hand-picking plugins to help ensure the success of the website.
    • Using JavaScript to customise the functionality of the website.
    • Choosing and editing a suitable design and theme for the website.
    • Managing social media pages and creating engaging content.
    • Using insights and Google analytics to further research the audience.
    • Integrating InkThreadable to automatically process orders.
    • Using Facebook to provide adverts to the target audience.

Skills Gained

  • How to independently learn - Through the making of the website I had to explore new topics that I had never come across. This included creating WordPress websites and marketing, which I found out by researching on Google, YouTube and taking online courses on CodeAcademy.
  • Self-discipline - Previously I had been working to deadlines set by other people, but in this case, I had to set milestones and goals to ensure that I created the website and content in a reasonable timeframe.

  • Patience and attention to detail - When working with users it is essential that there are minimal bugs. However, this can be stressful as I found out. This project helped me handle both of these and learn to balance them to create a successful software product.
